# Context Builder Utils

Pure helper utilities used by context-builder agents/orchestrator.

- `token-estimator.util.ts`: estimated token counting
- `text-chunker.util.ts`: line-based chunk splitting
- `similarity.util.ts`: text similarity
- `path-resolver.util.ts`: normalized import path resolution
- `logger.util.ts`: structured timestamped logs

No domain orchestration logic is implemented here.
