# Utils Folder

Helper-only utilities used by the orchestrator and agents.

- `logger.util.ts`: structured log message builder.
- `path-resolver.util.ts`: deterministic path construction.
- `string-formatter.util.ts`: string formatting helpers.
- `template-builder.util.ts`: template section joining.
- `yaml-builder.util.ts`: compose YAML serialization.

No domain orchestration logic lives here.
