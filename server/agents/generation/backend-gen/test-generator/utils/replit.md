# Utils Folder

## Purpose
Pure helper utilities used by agents/orchestrator.

## Files
- `template-loader.util.ts`: AAA test block template
- `test-format.util.ts`: suite-to-string formatter
- `naming.util.ts`: deterministic file naming helpers
- `import-builder.util.ts`: framework/target import generators

## Call graph
Called by agents and orchestrator only.

## Import rules
No domain logic and no backend execution.
