# Utils Layer (L3)

Helpers only, no orchestration/business pipeline decisions.

- `type-mapper.util.ts`: primitive type mapping by ORM.
- `naming.util.ts`: case conversion helpers.
- `schema-normalizer.util.ts`: sanitize names/types and normalize schema values.
- `template-loader.util.ts`: resolve ORM template renderer.
- `code-formatter.util.ts`: normalize whitespace/newline formatting.
